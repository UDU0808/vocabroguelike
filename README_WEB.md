- 角色、怪物、Boss、道具大图转为 WebP，显著降低体积。
- 删除未被引用的重复 PNG 素材。
- `wordbank.json` 改为压缩 JSON，保留全部词库内容。
- 新增 `assets/README_MODULES.md`，说明当前保留/移除的资源模块。


## v23 更新
- 错误翻译击中普通怪物后，该怪物会进入 15 秒警觉状态，追踪范围至少提升到 2 格。
- 怪物移动逻辑改为“一个格子一个格子移动”，从当前格中心走到相邻格中心，避免斜向卡在对角线或墙角。
- 正确翻译命中怪物仍直接消灭；错误翻译仍会重新刷新回地图。


## v24 更新

- 怪物加速状态增加红色脉冲提示，不恢复 A/B/C/D/E 专属外观颜色。
- 整体 UI 改为科技感风格：菜单、设置面板、暂停面板、底部提示框均改成蓝青色科技面板。
